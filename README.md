Edge-Fog Flood Monitoring System

A real-time distributed flood monitoring system using Edge Computing, Fog Computing, and Machine Learning.

## Features

- Edge-based flood prediction per city
- Fog-level regional risk analysis
- Pre-alert propagation to neighboring cities
- Adaptive Emergency Reallocation (AER)
- Real-time simulation dashboard using Streamlit
- Cloud storage using SQLite database
- Performance metrics (latency, accuracy, resource transfers)

## ⚙️ Setup Instructions

### 1️⃣ Clone Repository
```bash
git clone https://github.com/your-username/repo-name.git
cd repo-name
```

### 2️⃣ Install Dependencies
```bash
pip install -r requirements.txt
```

### 3️⃣ Run Backend (Simulation)
```bash
python main.py
```

### 3️⃣ Run Dashboard (In another terminal as soon as you run backend)
```bash
streamlit run app.py
```
▶️ How to Use
- Click Start to begin simulation
- Click Pause to stop at current state
- Click Start again to resume
- Click Reset to restart simulation
